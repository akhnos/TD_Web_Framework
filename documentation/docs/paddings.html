Paddings
Paddings are used to make an element take extra space for positioning purposes. The following 6 sizes can be used:
  • padding-mini {padding: 2px;}
  • padding-small {padding: 4px;}
  • padding-medium {padding: 10px}
  • padding-semiLarge {padding: 15px;}
  • padding-large {padding: 20px;}
  • padding-huge {padding: 30px;}
Example
By default paddings are put on all sides of an element. Adding -horizontal adds only right and left paddings and adding -vertical adds only top and bottom paddings.
