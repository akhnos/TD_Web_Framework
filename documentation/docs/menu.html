Accordion Menu, Dropdown Menu
Accordion menu and dropdown Menu offer easy to use menu functionality.
Accordion Example
Usage:
<div class="accordion-menu" >
   <div class="accordion-node-click">
      <div class="accordion-menu-header" onclick="$(this).siblings('.accordion-menu-content').removeClass('display-none');">
         Accordion Header
      </div>
      <div class="display-none accordion-menu-content">
         Accordion Content
      </div>
   </div>
</div>
