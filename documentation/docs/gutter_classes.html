Gutter Classes
Gutter classes are used to evenly seperate elements on vertical and horizontal plane.
Gutter Vertical Example
gutter-absolute-symmetric-medium-vertical
dh-0 dw-24
dh-0 dw-12
dh-0 dw-12
Gutter Horizontal Example (Dynamic)
By default horizontal placeholders are used. Note that with medium keyword space between segments equals to 1 unit.
gutter-symmetric-medium
dh-0 dw-23
dh-0 dw-11
dh-0 dw-11
dh-0 dw-5
dh-0 dw-5
dh-0 dw-5
dh-0 dw-5
Gutter Horizontal Example (Absolute)
An absolute value can be set using absolute keyword.
gutter-absolute-symmetric-high
dh-0 dw-23
dh-0 dw-11
dh-0 dw-11
dh-0 dw-5
dh-0 dw-5
dh-0 dw-5
dh-0 dw-5
