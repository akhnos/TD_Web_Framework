Grid Base Layout
Grid Base Layout is a structure designed to position html elements. When using Grid Base Layout, the page is divided into 24 segments and 5 resolutions. The segments are used to define the size and different sizes for different resolutions can be set.


Basic Example
Here you can see how different width segments are placed using the dw- class. Notice the dol- syntax which is used to add a margin of the given size to the left. Similar to dol-, dl- stands for left and dt stands for top. Different from others, dot- stands for margin-top, and is calculated by pixels instead of segments.

Responsive Example
Adding -lg, -md, -sm, and -xs specifies the resolution where different sizes should be implemented. If not specified 1280 and above, -lg is 1280 to 1024, -md is 1024 to 768 and -sm is 768 to 512 and -xs is 512 and below.

Window Positioning
Window Positioning is used to position an element within its parent element.
Example
By default, window-center is relatively positioned, it does not allow any elements written after it to occupy the same line while any element written before it can be on the same line. window-right, window-bottom and window-top are absolutely positioned.

Paddings
Paddings are used to make an element take extra space for positioning purposes. The following 6 sizes can be used:
  • padding-mini {padding: 2px;}
  • padding-small {padding: 4px;}
  • padding-medium {padding: 10px}
  • padding-semiLarge {padding: 15px;}
  • padding-large {padding: 20px;}
  • padding-huge {padding: 30px;}
Example
By default paddings are put on all sides of an element. Adding -horizontal adds only right and left paddings and adding -vertical adds only top and bottom paddings.

Accordion Menu, Dropdown Menu
Accordion menu and dropdown Menu offer easy to use menu functionality.
Accordion Example
Usage:
<div class="accordion-menu" >
   <div class="accordion-node-click">
      <div class="accordion-menu-header" onclick="$(this).siblings('.accordion-menu-content').removeClass('display-none');">
         Accordion Header
      </div>
      <div class="display-none accordion-menu-content">
         Accordion Content
      </div>
   </div>
</div>
