Grid Base Layout
Grid Base Layout is a structure designed to position html elements. When using Grid Base Layout, the page is divided into 24 segments and 5 resolutions. The segments are used to define the size and different sizes for different resolutions can be set.


Basic Example
Here you can see how different width segments are placed using the dw- class. Notice the dol- syntax which is used to add a margin of the given size to the left. Similar to dol-, dl- stands for left and dt stands for top. Different from others, dot- stands for margin-top, and is calculated by pixels instead of segments.

Responsive Example
Adding -lg, -md, -sm, and -xs specifies the resolution where different sizes should be implemented. If not specified 1280 and above, -lg is 1280 to 1024, -md is 1024 to 768 and -sm is 768 to 512 and -xs is 512 and below.
dh-0 dw-12 dw-18-md dw-24-xs
dh-0 dw-12 dw-6-md dw-24-xs
dh-0 dw-6 dw-24-md
